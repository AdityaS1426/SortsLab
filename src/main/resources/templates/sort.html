Insertion

First, the Array is split into a "sorted" & a "unsorted" section. Second, as the "sort" loops through each element, the elements are moved from the unsorted section into the sorted section.

Time

Best Sort Scenario: O(n)

Worst Case Scenario: O(n²)

Bubble

The Bubble Sort looks at the elements in the Array two at a time to determine whether swapping is necessary. If it is, a swap occurs. This pattern takes place until no two elements need to be swapped.

Time

Best Case Scenario: O(n)

Worst Case Scenario: O(n²)

Selection

Like the Insertion Sort above, the Selection Sort split the Array into a "sorted" & a "unsorted" section. The "sort" looks through the whole Array for the smallest value, moves it to the front of the Array, & repeats the process. The process is finished when the Array is ordered from least to greatest.

Time

Best Case Scenario: O(n)

Worst Case Scenario: O(n²)
